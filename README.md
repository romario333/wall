# wall
> Mobile-friendly list of climbing routes at Smichoff

This is just a prototype, you can see it in action here: https://romario333.github.io/wall/

## How to develop

```bash
# install dependencies
yarn install

# run development server
npm start

# build static site
npm run build
```
